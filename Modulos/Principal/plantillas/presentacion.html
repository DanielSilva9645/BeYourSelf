Sobre nosotros

Beyourself es un programa web que administra las actividades lúdicas dentro de la institución. Esta herramienta esta pensada para brindar apoyo a los docentes de las areas artisticas facilitandoles organizar su programación para clases

Ademas de eso, Beyourself cuenta con un sistema de inventarios que le permite a los usuarios solicitar materiales para su uso con fines educativos

Y tiene una interfaz con cualidades de una red social, que permite tanto hacer publicaciones como ver las de otras personas, creando una comunidad que comparte sus gustos y experiencias para cada uno de nuestros usuarios.
